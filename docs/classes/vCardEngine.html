Class vCardEngine

vCardEngine parses vCard strings and deduplicates lists of vCards.

Remark

The easiest way to parse a vCard string and get an array of cards:

Example

const engine = new vCardEngine(vcard_data);
let cards = [...engine.run()];

cards === vCard[]
